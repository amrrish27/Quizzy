# 🎯 Quizzy

Quizzy is an interactive quiz application developed as part of the CODSOFT Android App Development Internship.

## Features

* Multiple quiz categories
* Real-time score tracking
* Interactive question system
* Modern user interface
* Mobile-friendly design
* Firebase authentication support

## Technologies Used

* Next.js
* React
* TypeScript
* Firebase
* CSS

## Installation

```bash
git clone https://github.com/amrrish27/Quizzy.git
cd Quizzy
npm install
npm run dev
```

## Usage

1. Launch the application.
2. Sign in using the available authentication options.
3. Select a quiz category.
4. Answer questions and view your score.

## Project Objective

The objective of Quizzy is to provide an engaging learning experience through interactive quizzes while demonstrating modern frontend development practices.
